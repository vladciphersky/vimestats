# VimeStats
Discord бот для проверки статистики с VimeWorld

## Команды бота
- `help` | Помощь по командам
- `ping` | Проверить пинг клиента
- `staff` | Онлайн модераторов
- `stats <никнейм>` | Статистика пользователя
- `streams` | Стримы на сервере
